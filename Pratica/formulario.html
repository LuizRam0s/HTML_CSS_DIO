Exemplo praticado de formulario

As tags <form> são utilizadas para criar controles interativos para formulários baseados na web para receber dados do usuário, elas precisam ser fechadas.

A tag form aceitam alguns atributos como name, method, action, target, onsubmit e autocomplete

As tags abaixo usado como exemplo fica no meio das tags form citato acima.
