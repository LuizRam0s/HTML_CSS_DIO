Atributos com tabela

Pagina sobre alguns atributos, para fins didaticos e praticos

Atributo Descrição Tags que usam
align defini o alinhamento horizontal <hr> <img> <table> <td> <th> <tr>
border Defini a largura da borda. Utilizado com CSS <img> <table>
charset Declara a codificação dos caracteres da página ou do script <meta>
class usado com css, para estilizar os elementos com propriedades comum global atributo
color define a cor de um texto <font> <hr>
id global atributo usado com css, para estilizar os elementos com propriedades comum, esse valor deve ser unico
src URL de um conteúdo incorporável. <img> <video> <audio>
name nome do elemento <button> <form> <input> <meta>
method define o metodo HTTP ao enviar um formulario <form>
action a URI de um programa que processa a informação submetida através do formulário. <form>
target especifica onde abrir o documento vinculado pelo elemento de âncora <target>
onsubmit O valor do atributo é uma expressão JavaScript que será executada quando o formulário for enviado. usado com JavaScript
autocomplete formulário podem por padrão ter seus valores completados automaticamente pelo navegador. <autocomplete> <input>
type Define o tipo de elemento <input><button><style>
